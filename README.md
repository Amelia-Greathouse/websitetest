## To Save Changes
```
git add -A
git commit -m
git push
```
## To Deploy Website
```
mkdocs gh-deploy
```

## To make new tab in website (i.e. About me etc.)
### add new entry to mkdocs.yml -> nav list
```
-example: example/index.md
```
### in docs create new folder, then create new file in folder titled index.md
